Page description

The page description is the descriptive text associated with the page to better identify it in the report

Page ID

The page ID is a unique code assigned to the page by failover

Page Status

The page status indicates whether the page is prime, backup or unregistered

The register command adds the page as a condidate for prime selection

The unregister command removes the page from consideration for prime selection

The go prime command forces the current page to be selected as prime

The go backup command forces the current page to be unselected as prime

The rogue prime command sets the current page to prime, an invalid condition that the failover logic must address.

The report command lists the status of all pages managed by failover
